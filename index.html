HTML-CSS

El desarrollo frontal (frontend) es la creación de la interfaz y experiencia de usuario de un sitio o aplicación web usando tecnologías como HTML, CSS y JavaScript.

Comenzar

Temática:

Desarrolla interfaces de usuario efectivas utilizando HTML, CSS y JavaScript. Implementa diseños web responsivos que se adapten a diferentes dispositivos y resoluciones de pantalla. Además, aprende técnicas avanzadas de desarrollo front-end para crear sitios web modernos y dinámicos.

Temas:

  • Marcado web básico
  • Accesibilidad
  • Encontrabilidad (findability)
  • Marcado web avanzado.
  • Responsive WebDesign: Uso de media queries.
  • Frameworks de Diseño Web como "BootStrap v4".
  • Uso y generalidades de jQuery
